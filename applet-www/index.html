Applet RPM Repository

What is an Applet?

Applets (or Dock Apps) are small applications designed for use with AfterStep, WindowMaker, BlackBox, PWM, or Fluxbox.

What is this Project about?

Applets are rarely packaged with Linux distributions as most of the effort of distributions are spent on GNOME and KDE.  This project is designed to build both source and binary rpms for major Linux distributions so that users can easily install applets on their machines.

What is here so far?

  • Source RPMS - ftp://ftp.berlios.de/pub/appletrpms/srpms
  • Mandrake 9.1 i586 RPMS - ftp://ftp.berlios.de/pub/appletrpms/rpms

Also check out the Applet RPMs Project Page for more information.  I'll try to keep that up to date as well.

But wait, Applet YYY isn't there yet, why not?

Time is a limitted resource.  I started with the applets I needed on my new laptop, and will work through the applets that I most often use.  Please let me know what other applets your would like by registering a bug.  Make sure to set the category to "new applet request".

Did you write these applets?

No.  I'm just harvesting applets from the AfterStep Applets Distribution and Dock App Warehouse, and turning them into rpms.  Please see the README files in each rpm for more information about the authors of each individual applet.

Will there be more info here soon?

Definitely.  I just wanted to get something up here so that people can start playing with the applets and enjoying them.  Hope you enjoy!
